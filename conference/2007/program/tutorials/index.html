The AOSD 2007 tutorial program provides an in-depth opportunity to learn about state-of-the-art efforts in aspect-oriented software develpoment and programming. This year's tutorials are focused on topics relevant to those with an intermediate and advanced knowledge of aspect-oriented technology.

If you have any questions about the tutorial program, please contact the AOSD.07 Tutorial Chair at tutorialsaosd.net.
