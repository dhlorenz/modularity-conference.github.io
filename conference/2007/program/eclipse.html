Eclipse Event

Building AOSD Tools on Eclipse: Frameworks, APIs, and experiences

Wednesday, March 14, 8-10pm, Orca Room

This gathering is meant for those building or interested in building AOSD tools on Eclipse. We will start with presentations from the creators and integrators of frameworks that support AOSD. Each presentation will overview a framework, walk through example extensions, and discuss the Eclipse integration facilities that it provides. After the presentation we will have a hands-on session where the presenters will answer questions, and assist attendees with creating or improving their AOSD plug-ins. Bringing a laptop with an Eclipse workspace is not necessary, but encouraged for those interested in some hands-on coaching.

Agenda:

  • Introduction: Mik Kersten
  • Building on AspectJ: Adrian Colyer: presentation, code
  • Experiences extending AJDT: Charles Zhang presentation
  • Bytecode Engineering with ASM: Eugene Kuleshov
  • Experiences building research tools: Andrew Eisenberg and Terry Hon presentation, overlay movie, equation editing movie
  • Questions, discussions, and code camp
