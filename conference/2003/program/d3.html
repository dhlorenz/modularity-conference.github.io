The AspectJ Development Tools Project - Developing with AspectJ in Eclipse

Dates:
  - Wed March 19, 14:00 - 15:30
  - Thu March 20, 11:00 - 12:30

Presenters:
  - Adrian Colyer (IBM UK)
  - Andy Clement (IBM UK)
  - Matthew Webster (IBM UK)

This demonstration will show how Eclipse can be used to develop aspect-oriented software using AspectJ. We will walkthrough a set of a typical project development scenarios, showing how AspectJ can be applied and along the way illustrating the features of the latest AJDT plugin for Eclipse. The plugin and the Eclipse IDE are both made freely available so everything shown can be tried out on your own development projects too.
