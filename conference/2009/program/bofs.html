Birds of a Feather ... Flock together!

Sign Up for a "Birds of a Feather" (BoF) session:

At the registration desk, a signup sheet will be available to indicate interest in hosting a BoF throughout the conference week. A dedicated room is available for BoFs and the signup sheet will provide slots for available room reservation times. Make plans to propose and attend a BoF to discuss various topics with those attendees having similar interests!

JBoss AOP
Kabir Khan
JBoss AOP Lead. JBoss, a division of Red Hat
Flavia Rainone
JBoss AOP Developer. JBoss, a division of Red Hat

Tuesday, March 3, 3:00 pm - 5:30 pm (break at 3:30 pm after BoF introduction)
Location – South Meeting Room

Kabir and Flavia from the AOP team will host this BoF. It will start off with a presentation covering some of the JBoss AOP features:

  • A quick introduction to JBoss AOP
  • Weaving: Instrumentation vs. Proxy Factories
  • JBoss AS 5 & JBoss AOP Integration
  • Dynamic AOP with Hot Deployment and APIs
  • Integration with the new OSGi capable JBoss AS 5 class loaders

Following the main presentation there will be some demos showing:

  • How to run a standalone AOP application
  • Running an instrumented application in JBoss Application Server
  • Using aspects for Microcontainer beans in JBoss Application Server without instrumentation
  • Using JBoss AOP with Tomcat

The BoF is meant to be informal with plenty of time to ask questions as we go along. We will finish with an overview of future plans for the project, and our users can discuss new features they would like to see in JBoss AOP throughout the meeting.
